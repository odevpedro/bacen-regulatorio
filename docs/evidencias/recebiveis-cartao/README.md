# Evidencias - Recebiveis de Cartao

## Escopo

Base documental para as regras de unidade de recebivel, gravame, interoperabilidade e
cancelamento/alteracao.

- unidade de recebivel
- prioridade unica
- saldo disponivel
- interoperabilidade SI/CI/NI
- cancelamento e alteracao
